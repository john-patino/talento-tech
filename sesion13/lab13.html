Soluciones de Ejercicios de Funciones en JavaScript

Ejercicio 1: Área de un Círculo

Esta función calcula el área de un círculo dado el radio. La fórmula utilizada es Área = π * radio².

function calcularArea(radio) {
    return Math.PI * radio * radio;
}

Ejemplo: Si el radio es 5, el área es aproximadamente 78.54.

Ejercicio 2: Conversión de Celsius a Fahrenheit

La función convierte grados Celsius a Fahrenheit usando la fórmula F = (C * 9/5) + 32.

function convertirCelsiusAFahrenheit(celsius) {
    return (celsius * 9/5) + 32;
}

Ejemplo: Si la entrada es 25°C, la salida será 77°F.

Ejercicio 3: Verificación de Número Par o Impar

Esta función verifica si un número es par o impar usando el operador módulo (%).

function esPar(numero) {
    return numero % 2 === 0;
}

Ejemplo: Si el número es 4, la función devolverá "El número es par".
