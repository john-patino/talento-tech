Funciones en JavaScript

Conoce la teoría y práctica de funciones en JavaScript.

¿Qué es una Función?

Una función en JavaScript es un bloque de código reutilizable que realiza una tarea específica. Las funciones nos ayudan a organizar el código, mejorar su legibilidad y simplificar el mantenimiento.

Estructura de una Función

La estructura básica de una función incluye la palabra clave function, el nombre de la función, los parámetros opcionales y el cuerpo de la función entre llaves:

function nombreDeLaFuncion(parametro1, parametro2) {
    // Código de la función
    return resultado;
}

Tipos de Funciones

1. Función Declarativa

function saludar(nombre) {
    return "Hola, " + nombre;
}

2. Función Expresiva o Anónima

const saludar = function(nombre) {
    return "Hola, " + nombre;
};

3. Función Flecha

const sumar = (a, b) => a + b;

Ejercicios Prácticos

Ejercicio 1: Multiplicación

Crea una función multiplicar que acepte dos números y devuelva su producto:

function multiplicar(a, b) {
    return a * b;
}

Ejercicio 2: Verificar Números Pares

Crea una función esPar que devuelva true si un número es par y false si es impar:

function esPar(numero) {
    return numero % 2 === 0;
}

Ejercicio 3: Conversión a Mayúsculas

Crea una función convertirAMayusculas que reciba una cadena de texto y devuelva la misma cadena en mayúsculas:

function convertirAMayusculas(texto) {
    return texto.toUpperCase();
}

Ejercicio 4: Factorial de un Número

Usa una función recursiva llamada factorial para calcular el factorial de un número:

function factorial(n) {
    if (n === 0 || n === 1) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}

Ejercicio 5: Calculadora

Implementa una función calculadora que pueda realizar operaciones básicas según el operador proporcionado:

function calculadora(a, b, operador) {
    switch (operador) {
        case '+':
            return a + b;
        case '-':
            return a - b;
        case '*':
            return a * b;
        case '/':
            return b !== 0 ? a / b : "Error: División por cero";
        default:
            return "Operador no válido";
    }
}
